## SET UP VENV
1. Created venv and added aliast to ~/.bashrc
```bash
python3 -m venv project_venv
```
2. aliast to ~/.bashrc
```bash
alias myvenv='source <path to project_venv>/bin/activate'
```

3. install pre-commit
```bash
python3 -m pip install pre-commit
```
